# 闪烁的灯泡

*实数鸡*被*虚数鸡*关进了一个神秘的房间。这个房间的墙壁上安装了 $n$ 个灯泡和 $n$ 个开关，每个灯泡和开关都按照 $1, 2, \cdots, n$ 的顺序编号。在房间的天花板上，写有 $k$ 个神秘的数字：$a_1, a_2, \cdots, a_k$，其中 $\{a_1, a_2, \cdots, a_k\} \subset \{1, 2, \cdots, n\}$。

每当实数鸡按下编号为 $j$ 的开关时，编号为 $j + a_i \mod n$（其中 $i = 1, 2, \cdots, k$）的灯泡状态会发生改变：原本亮着的灯泡会变灭，而原本灭着的灯泡会变亮。如果所有灯泡都亮起，房间的门就会自动打开，实数鸡便能成功逃脱。如果存在一组数字 $\{a_i\}_{1 \leq i \leq k}$，使得无论灯泡的初始状态如何，实数鸡都能通过操作这些开关使所有灯泡亮起，那么这样的数字集合 $\{a_i\}_{1 \leq i \leq k}$ 被称为“好集合”。

**问题：**

固定正整数 $n$，尝试求解所有可能的“好集合”的数量。
